# 票迹 · 国内机票比价公开演示版

这是一个完全使用模拟数据的国内机票比价网页，用于展示航班搜索、筛选、排序和多平台报价比较交互。

## 功能

- 按出发城市、到达城市、日期和乘机人数搜索
- 按航空公司、起飞时段、机场和价格区间筛选
- 按推荐、最低价格、最早出发和最短耗时排序
- 展开携程、飞猪、同程和航司官网模拟报价
- 模拟购买跳转与全站演示数据提示
- 桌面端和移动端响应式布局

## 本地运行

直接用静态文件服务器打开根目录即可：

```bash
npx serve .
```

构建 Sites 部署包：

```bash
npm run build
```

> 演示数据，实际价格以平台为准。
